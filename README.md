Sistema de Agendamento para Barbearia

Um sistema web desenvolvido para facilitar o agendamento de horários em barbearias, permitindo que clientes marquem seus cortes de forma prática e que o barbeiro gerencie os atendimentos.

-------------------------------------------------------------------

O sistema simula um ambiente real de agendamento, onde:

Clientes podem inserir seu nome, escolher data e horário.
O sistema registra e armazena os agendamentos.
O barbeiro pode visualizar e gerenciar os horários marcados.
